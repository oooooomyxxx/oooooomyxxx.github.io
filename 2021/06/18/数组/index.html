数组

容器（放东西的东西）

  • 所元素具有相同的数据类型
  • 一旦创建，不能改变大小
  • 紧密依次排列
  • 下表从0开始计数，防止数组越界[0,数组大小-1]

函数

函数是一块代码，接受0个或多个参数，做一件事，并返回0个或1个值

  • ruturn停止函数的执行，并返回一个值
  • return；
  • return 表达式；
  • 一个语句中可以出现多个return语句

从函数中返回值

  • 可以赋值给变量
  • 可以再传递给函数
  • 丢弃不用

没有返回值的函数

  • void 函数名 （参数表）
  • 可以没有return
  • 如果用return，return不能带值
  • 调用的时候不能做返回值的赋值

原型声明

  • 函数头；
  • 告诉编译器这个函数的
    • 名称 ，
    • 参数（数量及类型），
    • 返回类型
